 E-Commerce Sales Analysis Using SQL
This project analyzes an e-commerce dataset using SQL to uncover customer behavior, 
product performance, sales trends, and revenue insights. 
The analysis includes data cleaning, standardization, handling missing values, 
and duplicate removal before performing exploratory data analysis (EDA).

Key analyses include:

* Customer analysis (top customers, repeat customers, inactive customers, customer lifetime value)
* Product analysis (top-selling products, revenue by product, category-wise sales)
* Sales and order trends (orders per day/month, average quantity per order)
* Revenue analysis (total revenue, revenue by day/month, revenue by product and category)
* Advanced SQL concepts such as CTEs, aggregations, joins, subqueries, and business KPI calculations.

This project demonstrates practical SQL skills for data analytics and business intelligence, transforming raw transactional data into actionable insights.
